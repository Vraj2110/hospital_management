# 🏥 Hospital Management System (Java Mini Project)

## 📌 Description

This is a simple **Hospital Management System** developed using core **Java and Object-Oriented Programming (OOP)** concepts. The project simulates basic hospital operations such as managing patients, doctors, and booking appointments through a console-based interface.

It is designed as a beginner-friendly project to strengthen understanding of Java fundamentals and prepare for real-world application development, including Android apps.

---

## 🚀 Features

* Add new patients
* Add new doctors
* Book appointments between patients and doctors
* View all appointments
* Simple menu-driven interface

---

## 🧠 Concepts Used

* Classes and Objects
* Encapsulation
* ArrayList (Dynamic Data Storage)
* Methods and Control Statements
* Basic User Input using Scanner

---

## 🛠️ Technologies

* Java (Core Java)
* IntelliJ IDEA / VS Code

---

## 📂 Project Structure

* `Patient.java` → Stores patient details
* `Doctor.java` → Stores doctor information
* `Appointment.java` → Links patient and doctor
* `HospitalManager.java` → Handles system logic
* `Main.java` → User interface (menu-driven program)

---

## 🎯 Purpose

This project is built to:

* Practice OOP concepts in Java
* Understand how real-world systems are designed
* Build a strong foundation for **Android App Development**

---

## 🔮 Future Improvements

The program runs in the console and allows users to interact using a menu system to manage hospital data.
